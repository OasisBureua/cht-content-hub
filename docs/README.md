# docs/

Repo-local documentation.

- `architecture/` — architecture decision records (ADRs) and high-level system documentation
- `runbooks/` — operational runbooks (how to debug a failing Lambda, how to roll back a deploy, how to recover from a DMS replication break)
- `conventions/` — code conventions, naming conventions, PR conventions

Cross-repo and stakeholder-facing documentation lives in Confluence.
