# docs/

Repo-local documentation.

- `architecture/` — architecture decision records (ADRs) and high-level system documentation specific to `cht-content-hub`
- `runbooks/` — operational runbooks (debugging a failing Lambda, rolling back a deploy, recovering from a state issue)
- `conventions/` — code conventions, naming conventions, API design rules

Cross-repo and stakeholder-facing documentation (migration plan, domain research, architecture proposal) lives in Confluence.
